# CLAUDE.md — рабочие правила (хакатон)

## Артефакты: абсолютные пути (КРИТИЧНО)
**Все артефакты (RESULTS.md, NOTES.md, README, демо-видео, скрипты-выводы) сохраняй по абсолютному пути в папке проекта:**

```
d:\GitHub\hackaton\          ← корень проекта, откуда запускается агент (его видно в редакторе)
  └─ sandbox-artifacts\      ← зеркало результатов тестов (RESULTS.md + *-NOTES.md)
```

Правила:
1. Любой файл, который человек должен увидеть, кладётся в `d:\GitHub\hackaton\` (а НЕ в `~`/home). Дублируй в проект сразу после создания.
2. Одноразовый sandbox-код может жить в `C:\Users\Вадим\hack-sandbox\`, но его **итоговые** заметки/результаты зеркаль в `d:\GitHub\hackaton\sandbox-artifacts\`.
3. В конце каждого блока пиши абсолютный путь к тому, что создал.
4. Не полагайся на `~` — на чужой машине/в чужом home утром в воскресенье это потерянный README.
